# liri-node-app

The liri-node-app take has several different functions that take user input and return information

node liri.js concert-this <artist/band name here> returns information on the artist/band's next event using the bandsintown api

node liri.js spotify-this-song <song name here> returns information about the song from spotify using the node spotify package

node liri.js movie-this <movie title here> returns information about the movie using the omdb api

node liri.js do-what-it-says calls the specified function and inserts the specified input that are both stored in a random.txt file


This project is a class assignment.
